# Store
## Description
<p>
  This is a back-end for the online store, the main product of which is different electronic devices.
</p>

## Tech stack
<ol>

  <li>MySQL</li>
</ol>

### Nest: 
<ul>

</ul>

### MySQL:
<ul>
  <li>Relative table structure</li>
  <li>Experience in MySQL</li>
</ul>
